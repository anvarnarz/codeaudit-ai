**Props interface:**

interface ThemeToggleProps {

**State and logic:**
- `const [theme, setTheme] = useState<"dark" | "light">("dark")`
- `useEffect` on mount: read `localStorage.getItem("theme")`, if `"light"` set state to `"light"`, else `"dark"`
- `toggleTo(t: "dark" | "light")` function:
  1. `setTheme(t)`
  2. `localStorage.setItem("theme", t)` — THIS IS THE DSYS-03 PERSISTENCE
  3. If `t === "dark"`: `document.documentElement.classList.add("dark")`
  4. If `t === "light"`: `document.documentElement.classList.remove("dark")`

**Render — segmented sun/moon control:**

**handleSubmit function:**
```typescript
async function handleSubmit() {
  setError("");
  setLoading(true);
  try {
    const result = await addApiKey(provider, apiKey, label);
    if (!result.success) {
      setError(result.error);
      setLoading(false);
      return;
    }
    await completeSetup(); // redirects to /dashboard
  } catch (e) {
    setError(e instanceof Error ? e.message : "Something went wrong");
    setLoading(false);
  }
}
```
